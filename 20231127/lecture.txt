제이쿼리 (j + Query)
자바스크립트의 라이브러리 중 하나

- 호환성 문제 (DOM, 객체)
- 애니메이션 구현
- **더 짧은 코드**

제이쿼리 사용하기

1. 다운로드 방식 (속도가 빠름)
jquery.com 에 접속해서 제이쿼리를 사용할 수 있는 코드를 다운로드 방법


2. 네트워크 방식 (용량이 줄어든다)
https://code.jquery.com/jquery-3.7.1.min.js

.js 파일 -> 제이쿼리를 자바스크립트로 구현한 코드
.min.js 파일 -> min(minimum 최소) -> 들여쓰기, 변수이름,
줄바꿈, 주석 -> 전부 최소화
-> .js 파일 대비 로딩이 빠름, 용량이 1/4 작음

=> <script src="주소">


제이쿼리를 사용하기 위해서는 제이쿼리를 불러오는(import 해오는)
코드가 반드시 script 들 중에서 가장 위에 있어야 함

document.querySelector 대신에 $ 하나로 대체 가능

JSON: Java Script Object Notation 구조
-> 데이터를 저장하거나 전송할 때 주로 사용하는 데이터 교환 방식
-> 객체를 표현할 때에도 사용

JSON 형식은 자바스크리트에서만 사용하는 것이 아니라 다른
언어들에서도 사용

제이쿼리에서 배열을 다루는 함수들
$.inArray -> 배열에 저장된 데이터 중에서 특정 데이터를 찾아서
인덱스 값을 반환

$.isArray -> 이 데이터가 배열인가 아닌가 (true, false)

$.merge ->두 배열을 하나로 합침

jquery에서 배열을 순회할 때 each 함수를 사용하는데,
배열.each 방식
-> 배열에 정의되어있는 each함수를 사용

$.each(배열, 함수) 방식
-> 제이쿼리로 장석된 자바스크립트 forEach 방식
