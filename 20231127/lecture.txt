자바스크립트의 라이브러리 중 하나

- 호환성 문제 (DOM, 객체)
- 애니메이션 구현
- **더 짧은 코드**

제이쿼리 사용하기

1. 다운로드 방식 (속도가 빠름)
jquery.com 에 접속해서 제이쿼리를 사용할 수 있는 코드를 다운로드 방법


2. 네트워크 방식 (용량이 줄어든다)
https://code.jquery.com/jquery-3.7.1.min.js

.js 파일 -> 제이쿼리를 자바스크립트로 구현한 코드
.min.js 파일 -> min(minimum 최소) -> 들여쓰기, 변수이름,
줄바꿈, 주석 -> 전부 최소화
-> .js 파일 대비 로딩이 빠름, 용량이 1/4 작음

=> <script src="주소">

document.querySelector 대신에 $ 하나로 대체 가능

JSON: Java Script Object Notation 구조
-> 데이터를 저장하거나 전송할 때 주로 사용하는 데이터 교환 방식
-> 객체를 표현할 때에도 사용

JSON 형식은 자바스크리트에서만 사용하는 것이 아니라 다른
언어들에서도 사용

$.isArray -> 이 데이터가 배열인가 아닌가 (true, false)

html 태그 css 바꾸기
자바스크립트 방식
~~~.style.CSS속성 -> 가져오기
~~~.style.CSS속성 = 값 -> 설정

제이쿼리 방식
~~~.css(속성) -> 가져오기
~~~.css(속성, 값) -> 설정

html 태그 속성 값 바꾸기
자바스크립트 방식
~~~.getAttribute(속성) -> 가져오기
~~~.setAttribute(속성, 값) -> 설정

제이쿼리 방식
~~~.attr(속성) -> 가져오기
~~~.attr(속성, 값) -> 설정



속성 탐색 선택자
$(태그[속성]) -> 해당 속성이 있는 태그 전부
$(태그[속성=값]) -> 해당 속성이 값은 요소 전부
$(태그[속성^=값]) -> 값으로 시작
$(태그[속성$=값]) -> 값으로 끝
$(태그[속성*=값]) -> 값을 포함
$(":hidden"), $(":visible"), $(".selected"), $(".checked")

$(":contains(텍스트)")
-> innerText에 텍스트를 포함하는 요소

$(":has(태그)")
-> 해당 태그를 자식으로 가지는 부모 요소

$(태그).find(검색요소)
-> 태그의 하위 요소 중 검색요소를 가진 부모 요소

$(태그 배열).filter(검색요소)
$(태그 배열).filter(함수)


$(태그).closest(검색요소)
-> 태그의 상위 요소 중 검색요소를 가진 자식 요소

$(태그).not(선택자)
-> 해당 선택자가 아닌 나머지

$(태그).eq(숫자)
-> 요소를 만족하는 값들 중에서 몇 번째 (0부터)
